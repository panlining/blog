指针

Posted on 2016-10-05   |   In C   |  

指针变量是一种特殊的变量，他存储的内容只能是内存地址，比如存储另一个变量的地址。

指针变量加减n，就是指针变量存储的地址值加减n倍的指针变量所指向数据类型所占的字节数。

在使用指针变量时需注意：

  • 指针变量在定义的同时可以初始化，如：
    int *p = & num;
  • 不能使用常量给指针变量赋值
  • 指针变量不论指向什么样的数据类型，都占用相同大小的空间
  • 指针变量的类型必须与它指向的数据类型一样
  • 指针变量指的是p，而不是*p。定义指针变量时的int表示它是指向整型变量的指针
  • 在明确指针变量指向变量num之后，*p就是这真变量p所指向的那个变量，即num

指针和数组

指针和一维数组

编译器在为一个一维数组开辟存储空间的时候，总是比数组元素个数多一个，用来存储该数组的首地址（起始地址），通过数组名来访问，数组名是一个指针常量。

指针和二维数组

在C语言中，数组存储是以行序为主序的。

编译器在创建二维数组array时，除了分配6个数组元素的存储空间，还自动生成了3个存储空间，分别通过array，array[0]，array[1]来访问，它们都是指针常量。

定义一个指向数组的指针

前边已经知道，二维数组的数组名array2存储的是array[0]的地址，所以array2就是一个指向一维数组的指针

以前面二行三列的数组为例，定义一个指向一维数组的指针：
int (*p)[3];
其中中括号中的3需要和二维数组第二维的维度值相同。这里指针p是一个指向由3个整型元素构成的一维数组的指针。
对p进行初始化，让它指向array[0]
p = array; // 等价于 p = &array[0]

指针数组

指针数组就是由指针组成的数组。在指针数组中，数组元素的类型不再是基本类型，而是指针类型。

指向指针的指针是一种特殊的指针，它存储的是某一个地址的地址。
定义方法：
char p;
char *pp;
pp = &p;
